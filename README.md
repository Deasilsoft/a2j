# aoe2-record-handler

A tool for handling AoE2 records.

## Setup

1. Clone repository.

       git clone https://github.com/Deasilsoft/aoe2-record-handler.git

2. Install mgz in project root.

       pip install mgz

## Usage

    add.py <file> <content>
    parse.py <file> <command>
    remove.py <file>
